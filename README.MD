## <i class="fa-solid fa-crosshairs"></i> What is VANGUARD?

**VANGUARD** is a fully public, real-time geopolitical intelligence dashboard built on the **MERN stack**. It aggregates live global events тАФ conflicts, protests, disasters, military movements, cyber threats тАФ onto an interactive 3D globe, and uniquely converts those events into **AI-powered financial trading signals** in real time.

No login. No authentication. Everything is visible and usable by any visitor instantly.

### Why Does This Exist?

| Problem | Reality |
|---|---|
| **Information overload** | Analysts juggle 10+ tabs: Reuters, BBC, FlightRadar, Bloomberg, Twitter тАФ manually connecting dots |
| **No financial context** | Every OSINT tool shows *what* happened. None say *what to do* financially |
| **Too slow** | By the time intelligence is processed, markets have already priced it in |
| **Fragmented tools** | Geopolitical events, market data, flight tracking, cyber threats all live in separate platforms |

**VANGUARD** solves all four problems in a single, unified, zero-cost intelligence platform.

## <i class="fa-solid fa-layer-group"></i> Feature Specifications

### <i class="fa-solid fa-globe"></i> Feature 1 тАФ Live 3D Geopolitical Globe

The centrepiece of VANGUARD. A WebGL globe that **slowly auto-rotates on idle** and shows every active geopolitical event on Earth as glowing, colour-coded dots.

- **Auto-rotation** тАФ Globe spins at ~2 RPM on idle, stops on user interaction, resumes after 8 seconds
- **Glowing event dots** тАФ Each event plotted at real lat/lon. Dot size scales with severity. Colour maps to threat level
- **Pulsing red rings** тАФ `CRITICAL` events show a double CSS keyframe ring animation drawing immediate attention
- **Click-to-Brief** тАФ Clicking any country triggers Groq AI to generate a full intelligence brief in a slide-in side panel
- **2D Map toggle** тАФ Button switches between Globe.gl 3D view and Mapbox GL JS flat 2D map with smooth transition
- **Event type filters** тАФ Toggle buttons: `Conflicts` `Protests` `Disasters` `Earthquakes` `Wildfires` `Military Flights`
- **Time range filter** тАФ Dropdown: `Last 1h` / `6h` / `24h` / `7 days` тАФ filters globe dots by event timestamp
- **Heatmap layer** тАФ Deck.gl density heatmap overlay showing event concentration per region. Toggle on/off
- **Country boundary glow** тАФ Hovering over a country highlights its boundary polygon in the country's severity colour
- **Fly-to animation** тАФ Clicking a news headline or ticker item smoothly flies the globe camera to that event location
- **Arc layer** тАФ Military flight paths rendered as animated arcs on the globe surface. Toggle independently

**Severity Colour System applied to all dots and rings:**

### <i class="fa-solid fa-newspaper"></i> Feature 2 тАФ Live Intelligence News Feed

Real-time, severity-coded news panel aggregating **150+ global sources**. Auto-refreshes every 5 minutes.

- **Multi-source aggregation** тАФ NewsAPI.org (100 req/day free) + direct RSS feeds via `rss-parser` npm from Reuters, BBC, Al Jazeera, Bloomberg
- **GDELT event feed** тАФ Real-time geopolitical events with coordinates. Unlimited, no key needed
- **Severity colour coding** тАФ Every headline auto-tagged using backend NLP keyword scoring: `CRITICAL` / `HIGH` / `MEDIUM` / `LOW`
- **Auto-refresh** тАФ Feed silently refreshes every 5 minutes. New items appear at top with fade-in animation
- **`BREAKING` badge** тАФ Stories covered by 3+ sources within 15 minutes get a pulsing gold `BREAKING` badge
- **Region filter tabs** тАФ `Middle East` `Europe` `Asia-Pacific` `Americas` `Africa`
- **Keyword search** тАФ Real-time search across all headlines. Matching terms highlighted in `#00D4FF` Cyber Cyan
- **Entity linking** тАФ Country names in headlines are clickable тАФ zooms globe camera to that country via `Globe.gl flyTo()`
- **Country flag on each headline** тАФ Every headline shows the country flag using `flagcdn.com` image (e.g. `https://flagcdn.com/w20/us.png`). Flag sits to the left of the headline title inline
- **Source badges** тАФ Each headline shows source name with Font Awesome icon: `fas fa-rss` for RSS sources, `fas fa-globe` for NewsAPI sources

### <i class="fa-solid fa-robot"></i> Feature 3 тАФ AI Country Intelligence Briefs (Groq)

Click any country on the globe тЖТ a slide-in panel appears with a full AI-generated intelligence brief in under 2 seconds.

- **Groq Llama 3.1 8B** тАФ Primary AI model. Generates 3-paragraph brief using live headlines as context
- **4-key rotation system** тАФ 4 separate free Groq API keys (`GROQ_API_KEY_1` through `GROQ_API_KEY_4`) used in round-robin. If key 1 hits rate limit, key 2 activates instantly. Key 3 next, then key 4. This ensures the AI never visibly fails during a demo
- **Google Gemini fallback** тАФ If all 4 Groq keys are exhausted, Gemini API (15 req/min free) handles the request seamlessly
- **Stability Score** тАФ AI assigns a 0тАУ100 score rendered as a glowing `Recharts RadialBarChart` ring. Red if <30, Orange if 30тАУ60, Green if >60
- **Top 3 risks** тАФ AI returns the three most critical current risk factors as structured bullet points
- **Outlook badge** тАФ `Stable` / `Deteriorating` / `Escalating` / `Crisis` тАФ colour-coded badge with Font Awesome icon: `fas fa-chart-line`
- **Source citations** тАФ Brief lists exactly which headlines were used as context (transparent AI)
- **15-minute MongoDB cache** тАФ Generated briefs cached with a TTL to prevent rate limit abuse
- **Country metadata** тАФ Flag, capital city, population, government type from `RestCountries API` (free, unlimited). Country flag displayed prominently using `flagcdn.com/w40/{iso2}.png`
- **Share button** тАФ `fas fa-copy` icon copies brief to clipboard via Clipboard API

### <i class="fa-solid fa-chart-line"></i> Feature 4 тАФ GeoTrade AI Signal Engine

**The core differentiator.** The AI reads live geopolitical events and generates real-time `BUY` / `HOLD` / `SELL` signals for stocks, crypto, forex, and commodities.

- **Stock search** тАФ Input field with `fas fa-search` icon. Accepts any ticker: `AAPL` `TSLA` `XOM` `GLD` `SPY`
- **Live price & change** тАФ Real-time price, % change, volume via `yahoo-finance2` npm (free, no API key). Green `fas fa-arrow-up` for positive, Red `fas fa-arrow-down` for negative
- **7-day sparkline chart** тАФ Recharts `LineChart` sparkline for past 7 days price history. Cyan line on dark panel background
- **AI signal generation** тАФ Groq reads current GDELT events related to the stock's sector/country тЖТ outputs `BUY` / `HOLD` / `SELL` with full reasoning paragraph. 4-key rotation applies here identically
- **Signal confidence** тАФ Each signal shows a confidence percentage (e.g., `BUY тАФ 81%`) displayed in JetBrains Mono font
- **Signal badges** тАФ Glowing coloured badges with Font Awesome icons:
  - `BUY` тАФ `fas fa-arrow-trend-up` тАФ green glow (`#00FF88`)
  - `SELL` тАФ `fas fa-arrow-trend-down` тАФ red glow (`#EF4444`)
  - `HOLD` тАФ `fas fa-pause` тАФ yellow glow (`#EAB308`)
- **Fear & Greed Index** тАФ `alternative.me` API dial widget showing global market sentiment (0тАУ100). Displayed with `fas fa-gauge-high` icon
- **Crypto panel** тАФ BTC / ETH / SOL / XRP live prices with coin logos via CoinGecko API (free). Geo-risk correlation labels
- **Forex panel** тАФ USD/JPY, EUR/USD, GBP/USD, USD/INR live rates. Country flags displayed for each currency pair using `flagcdn.com`
- **Commodity panel** тАФ Gold `fas fa-coins`, Crude Oil `fas fa-oil-well`, Natural Gas `fas fa-fire-flame-curved` via yahoo-finance2 with conflict-driven context labels
- **Signal history** тАФ Last 5 AI signals per ticker stored in MongoDB. Shown as a mini timeline below the current signal

### Country Flags

> **Agent Implementation Note:** Country flags appear in 4 places: news feed headlines, country brief panel header, forex currency pairs, and the tension ticker. Use `flagcdn.com` тАФ it is free, no API key needed, and returns a PNG flag for any ISO 3166-1 alpha-2 country code.

```
Format: https://flagcdn.com/w{size}/{iso2_lowercase}.png

Sizes available: w16, w20, w32, w40, w80, w160, w320

Examples:
  US flag (20px wide): https://flagcdn.com/w20/us.png
  Ukraine flag:        https://flagcdn.com/w20/ua.png
  China flag:          https://flagcdn.com/w20/cn.png
  Iran flag:           https://flagcdn.com/w20/ir.png
```

**Implementation in React:**

```jsx
// FlagIcon component
const FlagIcon = ({ iso2, size = 20, className = '' }) => (
  <img
    src={`https://flagcdn.com/w${size}/${iso2.toLowerCase()}.png`}
    alt={iso2}
    width={size}
    className={`inline-block rounded-sm ${className}`}
    style={{ imageRendering: 'crisp-edges' }}
  />
);

// Usage in news headline:
<div className="flex items-center gap-2">
  <FlagIcon iso2="ua" size={20} />
  <span className="font-sans text-sm text-slate-300">{headline.title}</span>
</div>

// Usage in brief panel header:
<FlagIcon iso2={country.iso2} size={40} className="rounded" />

// Usage in forex pair:
<div className="flex items-center gap-1">
  <FlagIcon iso2="us" size={16} />
  <FlagIcon iso2="jp" size={16} />
  <span className="font-mono text-white">149.82</span>
</div>
```
